# Analysis of Graduate Employability Trends in Nigeria

## Project Overview
This project analyzes graduate employability trends in Nigeria using data visualization and dashboarding techniques in Microsoft Excel.

The objective was to identify how factors such as university type, field of study, GPA, and postgraduate qualifications influence employment outcomes and salary levels.

## My Contribution
This was a team Capstone Project.

I was part of the Dashboard Design & Visualization team, where I collaborated with team members to transform analyzed data into an interactive and easy-to-understand Excel dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Data Visualization

## Key Insights
- State university graduates recorded the highest average salary.
- Engineering had the highest underemployment rate.
- Postgraduate degrees showed minimal impact on early-career salary.
- First Class and Second Class Upper graduates earned similar average salaries.
