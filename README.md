# Graduate Employability Trends in Nigeria

## Project Overview

This project explores graduate employability trends in Nigeria using Microsoft Excel, with a focus on understanding how factors such as university type, field of study, academic performance (GPA), and postgraduate qualifications influence employment outcomes and salary levels.

The project was completed as part of the TechCrush Data Analytics Capstone Project and involved data cleaning, exploratory data analysis (EDA), dashboard development, and data storytelling.

---

## Business Problem

Many graduates and career advisors assume that factors such as attending a particular university type, earning a higher GPA, or obtaining a postgraduate degree automatically lead to better employment opportunities and higher salaries.

This project uses data to evaluate these assumptions and uncover the factors that truly influence graduate employability outcomes in Nigeria.

---

## Research Questions

The analysis was conducted to answer the following questions:

1. Do private university graduates earn more on average than federal university graduates?
2. Which field of study has the highest rate of underemployment?
3. Is there a salary ceiling for graduates without postgraduate qualifications?
4. Does a higher GPA significantly affect starting salary?

## Data Preparation and Cleaning

The dataset was prepared using Power Query in Microsoft Excel.

The following transformations were performed:

- Verified and corrected data types
- Corrected region classification inconsistencies using conditional logic
- Split datasets into multiple queries for efficient transformation

## Key Insights

### University Type and Salary

State university graduates recorded the highest average salary, outperforming graduates from both federal and private universities.

### Underemployment by Field of Study

Engineering graduates recorded the highest underemployment rate at **61.1%**, making it the most affected field in the dataset.

### Impact of Postgraduate Qualifications

The analysis showed that postgraduate qualifications had minimal influence on early-career salary outcomes.

### GPA and Salary

Graduates with First Class and Second Class Upper degrees earned very similar average salaries, suggesting that exceptional academic performance did not necessarily translate into significantly higher earnings.
